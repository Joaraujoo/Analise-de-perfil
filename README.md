# Análise de Perfil de Pessoas
Aplicação desenvolvida em Python para cadastro e análise de dados demográficos, utilizando listas e dicionários para armazenar informações como nome, sexo e idade, permitindo o cálculo de métricas e segmentação dos dados.

# Funcionalidades
- Cadastro de pessoas com nome, sexo e idade
- Validação de dados de entrada (sexo e continuidade do cadastro)
- Armazenamento estruturado das informações em listas e dicionários
- Cálculo da média de idade do grupo
- Identificação das pessoas do sexo feminino cadastradas
- Listagem das pessoas com idade acima da média
- Exibição organizada dos dados analisados

# O que é analisado
- Quantidade total de pessoas cadastradas
- Média de idade do grupo
- Segmentação por sexo
- Identificação de pessoas acima da média de idade
- Estruturação e consulta de dados demográficos

# Tecnologias Utilizadas
-Python 3
